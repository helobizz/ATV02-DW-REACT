# ATV02-DW-REACT

Atividade 02 — Consumo de API no React

Projeto desenvolvido com **React** e **Next.js** que consome a API pública [HP API](https://hp-api.onrender.com/), listando os personagens da saga Harry Potter.

## Funcionalidades

- Listagem de todos os personagens
- Página de detalhe de cada personagem

## Tecnologias utilizadas

- React
- Next.js
- CSS Modules

## Como rodar o projeto

```bash
# Instalar as dependências
npm install

# Rodar em modo de desenvolvimento
npm run dev
```

Acesse no navegador: `http://localhost:3000`

## API utilizada

[https://hp-api.onrender.com/](https://hp-api.onrender.com/)
